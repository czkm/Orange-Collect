
## Welcome to orange_collect 🧘

```
旨在收集有意思的按钮或相关效果，并封装成组件📦
```

### How to used

```
npm i orange_collect
```

在main.js 中

```
import orange from 'orange_collect'
import 'orange_collect/lib/orange_collect.css'

Vue.use(orange)
```
