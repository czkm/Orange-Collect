
## Welcome to orange_collect 🧘

```
旨在收集有意思的按钮或相关效果，并封装成组件📦
```

### How to used

```
npm i OrangeCollect -S
```

在main.js 中

```
import OrangeCollect from 'orange_collect'
Vue.use(OrangeCollect)
```
